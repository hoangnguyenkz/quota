Đồng bộ với Google Sheets

Ứng dụng lưu dữ liệu ngay trên trình duyệt. Để chia sẻ dữ liệu cho nhiều người cùng dùng, kết nối với 1 Google Sheet thông qua Google Apps Script (miễn phí, không cần server riêng).

Bước 1 — Tạo backend trên Google Sheets

1. Tạo 1 Google Sheet mới.
2. Vào Tiện ích mở rộng → Apps Script.
3. Xóa hết code mẫu, dán đoạn mã bên dưới (cũng có trong file "apps-script-backend.gs" đã được cung cấp kèm).
4. Bấm Triển khai → Triển khai mới → Web app. Chọn "Thực thi với: Tôi", "Người có quyền truy cập: Bất kỳ ai".
5. Sao chép URL Web App, dán vào ô bên dưới.

Bước 2 — Kết nối
Chưa kết nối Google Sheets — dữ liệu chỉ đang lưu trên trình duyệt này.
